# Codigo de conducta

Este proyecto busca mejorar la accesibilidad a herramientas digitales para la comunidad juridica.

Se espera una participacion respetuosa, profesional y orientada a usuarios reales. No se aceptan contribuciones que ridiculicen a personas mayores, minimicen barreras de accesibilidad o promuevan automatizacion juridica sin control profesional.

Las discusiones deben enfocarse en evidencia, claridad, seguridad y utilidad publica.
