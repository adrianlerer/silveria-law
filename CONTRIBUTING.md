# Contribuir

Gracias por querer mejorar SilverIA Law.

## Buenas contribuciones

- Mejoras de accesibilidad.
- Correcciones de contraste, foco o navegacion por teclado.
- Textos mas claros para usuarios no tecnicos.
- Documentacion para talleres.
- Adaptaciones institucionales no propietarias.
- Pruebas con usuarios reales y hallazgos reproducibles.

## Reglas

- No subir datos personales, expedientes reales ni documentos confidenciales.
- No agregar prompts privados ni estrategias propietarias.
- No presentar la app como sustituto de asesoramiento juridico.
- Mantener el codigo simple y auditable.
- Priorizar claridad antes que automatizacion llamativa.

## Verificacion local

```bash
npm run check
```

Para probar la app:

```bash
npm run serve
```
